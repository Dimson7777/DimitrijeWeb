## Full Stack Developer Portfolio

This is a modern, production-ready developer portfolio built to showcase real-world projects, technical skills, and provide a seamless way for clients or recruiters to connect and schedule a call.

The goal of this project was not just to present projects, but to create a complete professional presence — combining strong UI/UX, real application logic, and a functional booking system.

---

## 🚀 Features

* Clean, modern, human-centered UI (no generic templates)
* Fully responsive design (mobile-first)
* Project showcase with live demos and GitHub links
* Personal branding section with strong introduction
* Smooth animations and polished user experience

### 📅 Booking System (Key Feature)

* Interactive calendar for scheduling calls
* Select date and available time slots
* User input (name, email, message)
* Booking confirmation flow
* Simulated backend logic for managing appointments
